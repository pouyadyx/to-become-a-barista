# Coffee Image Slots

Put your finished coffee photos in this folder using the filenames from `data/coffees.js`.

Each coffee has three slots:

- `<coffee-id>-1.jpg` is the main image for cards and detail pages.
- `<coffee-id>-2.jpg` and `<coffee-id>-3.jpg` are secondary detail photos.

Examples:

- `espresso-1.jpg`
- `latte-1.jpg`
- `spanish-latte-1.jpg`
- `dirty-1.jpg`
- `vietnamese-coffee-1.jpg`

If a file is missing, the site automatically shows `placeholder.svg` instead of a broken image.
